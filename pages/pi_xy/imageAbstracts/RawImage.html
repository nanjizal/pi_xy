abstract RawImage(Pixelimage)

package pi_xy.imageAbstracts

from Pixelimage to Pixelimage

@:transient

Provides methods to deal directly with the raw Pixelimage data take care as the internal colour values are not ARGB format methods are very similar to the Picture class within Img in iterMagic

Static methods

staticinlineclone(this:Pixelimage):Pixelimage

clone the current Pixelimage to a new one, does not copy the mask.

staticinlinefromTo(a:Pixelimage, b:Pixelimage):Pixelimage

transfers pixel values directly from first image to second returns second image

@:arrayAccessstaticinlineget(this:Pixelimage, index:Int):Int

staticinlineimgToString(this:Pixelimage):String

traces the pixel values stored internally

@:arrayAccessstaticinlineset(this:Pixelimage, index:Int, value:Int):Int

direct array access of pixel Int not advised as pixels normally have mask and blend applied and the color order is changed when set indirectly. see setPixel on Pixelimage

staticinlinetoFrom(a:Pixelimage, b:Pixelimage):Pixelimage

staticinlinetraceGrid(this:Pixelimage):Void
